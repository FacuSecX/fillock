**Fillock** es un simple script para android que sirve para encriptar archivos mediante OpenSSL, sirve basicamente como una aplicacion pero de manera ilimitada, el scrip puede ser utilizada en cualquier emulador como Termux y permite cifrar archivos, por extensiones, audios, videos, docummentos o imagenes.. o cifrado completo

**AVISO IMPORTANTE:** usar este programa es bajo su responsabilidad ya que tambien puede utilizarse de forma incorrecta y podria servir como **ransomware** tiene que tener
en cuenta que al momento de encriptar cualquier tipo de archivo, la clave o contraseÃ±a utilizada debe ser respaldada, si pierde esta contraseÃ±a, los archivos  son **irrecuperables**


## Funciones automatizadas incluidas:

| Funcion        |   Informacion                                                                                                                      |
|---------------|------------------------------------------------------------------------------------------------------------------------------------ |                                                                                
| Galeria Lock:                            | Cifra toda la galeria de /sdcard/                                                                        |
| Cifrado Completo                         | Cifra todo tipo de archivos del directorio raiz pueden agregar extensiones si desean                     |
| Filtrar cifrado por extensiones          | Filtra y cifra todos los archivos de la raiz por extensiones (videos, imagenes, audios o documentos)     |
| Eliminar Metadatos                       | Elimina cualquier metadato de archivos                                                                   |
| Cambiar nombres aleatorios a archivos    | renombra los archivos a nombres aleatorios para dificultar su identificacion                             |
